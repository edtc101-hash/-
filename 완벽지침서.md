# 완벽지침서: 브랜드 콘텐츠 제작 가이드

> 이 문서는 브랜드 콘텐츠를 기획하고 제작할 때 반드시 참고해야 할 핵심 지침입니다.
> 모든 콘텐츠는 아래의 원칙과 기준에 따라 제작되어야 합니다.

---

## 1. 핵심 철학 (WHY)

### 브랜드 목적
> "하나의 브랜드를 깊이 있게 해부하여, 일과 삶을 대하는 태도를 재정의한다."

### 브랜드 비전
비즈니스 종사자들이 의사결정의 순간마다 참고하는 **'브랜드 인사이트의 기준서'**가 되는 것.

### 3대 핵심 가치

| 가치 | 원칙 | 실행 기준 |
|------|------|-----------|
| **깊이 > 속도** | 트렌드 요약 ✕, 한 브랜드를 한 권에 담는 밀도 | 단기 클릭 유도가 아닌 장기 소장 가치를 추구할 것 |
| **존중 기반 객관성** | 찬양도 비판도 아닌 구조 분석 | 철학 · 조직 · 제품 · 재무를 균형있게 조명할 것 |
| **편집 장인정신** | 종이 질감, 타이포, 사진 톤까지 통일된 미감 | 콘텐츠를 '정보'가 아닌 '오브제'로 완성할 것 |

---

## 2. 타겟 & 포지셔닝 (WHO & WHERE)

### 타겟 독자
- **주요 대상:** 브랜드 전략, 마케팅, 디자인, 창업에 관심 있는 **3~10년 차 실무자**
- **핵심 니즈:** 브랜드를 소비하는 것이 아니라 **"설계"**하고 싶은 사람

### 경쟁 환경

| 구분 | 경쟁 대상 | 우리의 차별점 |
|------|-----------|---------------|
| 직접 경쟁 | 트렌드 리포트, 마케팅 뉴스레터 | 그들은 속도는 빠르지만 깊이가 제한적 |
| 간접 경쟁 | 기업 자서전, 창업 스토리 북 | 그들은 서사 중심이고 구조 분석이 부족 |

### 차별화 공식
```
한 브랜드 = 한 권 (포맷의 일관성)
연재 콘텐츠 ✕ → 아카이빙 가능한 '브랜드 다큐멘터리' ○
정보 소비 도구 ✕ → 브랜드 사고방식의 학습 도구 ○
```

## 3. 톤 & 보이스 (HOW WE SPEAK)

### 브랜드 아키타입
**The Archivist (기록자) + The Analyst (분석가)**

탄생 배경 → 조직 문화 → 제품 설계 철학을 차분히 기록하고, 감정이 아닌 **구조로 설명하는 관찰자**.

### 성격 3원칙

| DO ✅ | DON'T ❌ |
|-------|----------|
| 과장 없는 데이터와 인터뷰 기반 서술 | 감정적 찬사, 과대 포장 |
| 구조화된 문단으로 차분한 권위 형성 | 감탄사, 느낌표 남발 |
| 유행보다 지속성에 주목 | 단기 트렌드 추종 |

### 글쓰기 규칙
1. **서술 방식:** 묘사적이되 절제된 문체 (Descriptive but restrained)
2. **문장 구조:** 구조 중심의 문장. 핵심 → 맥락 → 근거 순서
3. **단어 선택:** 형용사 최소화, 사실과 맥락 중심으로 작성
4. **금지 표현:** "혁신적인", "놀라운", "역대급" 등 과장 형용사 사용 금지

## 4. 비주얼 아이덴티티 (HOW WE LOOK)

### 디자인 원칙

| 요소 | 기준 |
|------|------|
| **표지** | 미니멀, 단색 또는 절제된 컬러 |
| **레이아웃** | 그리드 기반, 정돈된 여백 |
| **인쇄** | 두께감 있는 종이, 촉감이 느껴지는 질감 |
| **전체 방향** | 책장에 꽂아두고 싶은 오브제성 |

### 비주얼 체크리스트
- [ ] 시각적 일관성이 유지되는가?
- [ ] '기록물'이라는 인식이 느껴지는가?
- [ ] 불필요한 장식 요소가 없는가?
- [ ] 그리드 정렬이 정확한가?

## 5. 독자 경험 설계 (WHAT WE DELIVER)

### 브랜드 약속
> "당신은 한 브랜드를 깊이 이해함으로써, 자신의 일에 대한 기준을 갖게 됩니다."

### 독자 여정 (Experience Journey)

```
1. 발견 (The Hook)
   → 관심 있는 브랜드가 단독 이슈로 다뤄졌다는 사실을 발견

2. 몰입 (The Immersion)
   → 창업 배경, 제품 철학, 조직 구조를 한 흐름으로 읽음

3. 성찰 (The Reflection)
   → "우리 조직은 어떤가?"라는 질문을 스스로에게 던짐

4. 적용 (The Application)
   → 배운 구조를 자신의 일에 적용
```

## 6. 최종 요약

| 질문 | 답 |
|------|----|
| 우리는 무엇을 하는가? | 정보를 전달하는 것이 아니라 **'브랜드를 읽는 법'을 교육**한다 |
| 카테고리는? | 매거진 |
| 실제 기능은? | **브랜드 사고 훈련서** |
| 콘텐츠 제작 시 최우선 원칙은? | **깊이 > 속도, 구조 > 감성, 소장 가치 > 클릭 유도** |
